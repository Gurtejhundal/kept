# Feature Scope

## MVP features

### Authentication
- Email and password
- Google sign-in
- Apple sign-in for iOS
- Optional username and profile image
- Account deletion

### Save flow
- Accept shared text from Instagram or clipboard
- Detect one or more URLs in shared content
- Ask for reel URL only when not available
- Allow thumbnail upload or automatic thumbnail extraction where technically possible
- Automatically store save date and time
- Automatically detect domain
- Generate suggested title
- Generate suggested category
- Let user edit title, category, notes, and collection
- Complete save in one confirmation screen

### Library
- Chronological home feed
- Today, this week, this month, and older sections
- Two-column visual card grid on mobile
- Grid/list toggle on web
- Sort by newest, oldest, recently opened
- Filter by category, creator, platform, collection, and status

### Search
- Full-text search
- Date range
- Creator
- Category
- Link domain
- Collection
- Deleted reel status
- Search recent history

### Item detail
- Thumbnail
- Title
- Original reel URL
- Creator name
- Destination URL
- Date and time received
- Date and time saved
- Category
- Notes
- Collection
- Tags
- Reel availability
- Open, copy, share, edit, archive, and delete actions

### Collections
- Create, rename, reorder, archive, and delete
- Add or remove saved items
- Private by default
- Optional share link
- Revoke shared link

### Reel status
- Periodic check for public reel availability
- Mark as active, unavailable, unknown, or deleted
- Preserve thumbnail and metadata even when unavailable

### Notifications
- Optional weekly resurfacing
- “You saved this one month ago”
- Failed metadata extraction
- Reel deletion alert for premium users

## Post-MVP features
- Duplicate detection
- Semantic search
- OCR from screenshots
- Multiple links per reel
- Bulk import from screenshots or notes
- Smart resurfacing
- Shared collaborative collections
- Cross-device clipboard capture
- Premium unlimited archive
- Link health checks
- Personal tags and custom fields

## Explicitly excluded from MVP
- Instagram account scraping
- Automatic reading of private DMs
- Browser extension
- Affiliate tracking
- Price and stock tracking
- Public social feed
- Creator campaign dashboard
