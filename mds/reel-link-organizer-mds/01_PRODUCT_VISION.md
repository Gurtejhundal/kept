# Product Vision

## Vision statement
Create a visual memory system for links discovered through Instagram, allowing users to save an item in seconds and find it later using the reel thumbnail, creator, date, text, or category.

## User promise
“Anything you discover through an Instagram reel can be found again when you actually need it.”

## Product positioning
ReelRecall sits between:

- Instagram inbox
- Bookmark managers
- Visual discovery tools
- Personal knowledge management
- Shopping and recommendation organizers

It is not meant to replace Instagram. It provides the organizational layer Instagram does not provide.

## Product goals
1. Reduce lost Instagram DM links.
2. Make saved items visually recognizable.
3. Support date-first and text-first retrieval.
4. Preserve context around why the link was saved.
5. Allow users to optionally share collections.
6. Create a subscription path through advanced organization and monitoring.

## Non-goals for the first release
- Building a social network
- Recreating Instagram reels
- Browser extension development
- Price tracking
- Product inventory tracking
- Public creator marketplace
- Affiliate commerce
- Full AI chat assistant
- Automatic access to private Instagram messages without explicit user action

## Success criteria
The first version is successful when:
- A user can save an Instagram DM link in under 10 seconds.
- A saved item includes a thumbnail, source, date, and destination.
- A user can retrieve an item by approximate date or keyword.
- The system remains useful even when the reel is later deleted.
- At least 35% of activated users save five or more items in their first week.
