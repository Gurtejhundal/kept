# Design System

## Visual theme
**Soft Editorial + Digital Scrapbook**

## Core colors
- Background: `#F6F5F1`
- Surface: `#FFFFFF`
- Primary text: `#181818`
- Secondary text: `#77746E`
- Border: `#E5E2DA`
- Primary accent: `#6C5CE7`
- Primary accent soft: `#EEEAFE`
- Action accent: `#FF654F`
- Success: `#278A66`
- Warning: `#C47A18`
- Error: `#C94848`
- Dark background: `#111111`
- Dark surface: `#1A1A1A`

## Typography
- Primary UI: Inter
- Display: Manrope or Plus Jakarta Sans
- Optional editorial date headings: Instrument Serif

## Type scale
- Display: 40/48
- H1: 32/40
- H2: 24/32
- H3: 20/28
- Body large: 17/26
- Body: 15/23
- Small: 13/18
- Label: 12/16

## Spacing
Use an 8-point base system:
- 4, 8, 12, 16, 24, 32, 40, 48, 64

## Radius
- Small controls: 10px
- Inputs: 14px
- Cards: 18px
- Large media cards: 22px
- Pills: 999px

## Shadows
Use subtle elevation only:
- Card: low-opacity soft shadow
- Modal: stronger but diffused
- Avoid heavy black shadows

## Iconography
- Lucide icons
- 1.75–2px stroke
- Rounded geometry
- No mixed icon styles

## Motion
- Standard transition: 180–240ms
- Modal/sheet: 260–320ms
- Save confirmation: subtle scale and fade
- No decorative motion that delays interaction
- Respect reduced-motion settings

## Card variants
- Standard saved card
- Large memory card
- Compact list row
- Incomplete metadata card
- Deleted reel card
- Shared collection card

## Status language
Use plain language:
- Reel available
- Reel unavailable
- Reel deleted
- Link unavailable
- Details incomplete

Avoid vague technical labels such as “fetch failed” in the consumer UI.
