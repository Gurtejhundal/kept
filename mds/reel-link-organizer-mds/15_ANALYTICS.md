# Product Analytics

## North-star candidate
Number of saved items successfully reopened after at least seven days.

This measures real retrieval value, not superficial engagement.

## Activation events
- Account created
- First save completed
- First thumbnail added
- First collection created
- First search performed
- First saved item reopened
- Five saves completed

## Core metrics
- Save completion rate
- Median time to save
- Metadata extraction success rate
- Search success rate
- Search-to-open rate
- Seven-day retrieval rate
- Weekly active savers
- Saves per active user
- Collection creation rate
- Shared collection rate
- 7-day and 30-day retention
- Reel deletion alert engagement

- `collection_shared`
- `subscription_started`

## Privacy rules
- Avoid sending full URLs to analytics providers.
- Hash or classify domains when possible.
- Never send personal notes or message text.
- Provide analytics opt-out where required.
